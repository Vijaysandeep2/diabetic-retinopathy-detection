# 👁️ Diabetic Retinopathy Detection using CNN

A deep learning project that detects diabetic retinopathy from retinal images using Convolutional Neural Networks with InceptionV3 architecture achieving 94% accuracy.

## 🚀 Tech Stack
- Python 3.10+
- TensorFlow & Keras
- InceptionV3 (Transfer Learning)
- NumPy & Pandas
- Matplotlib & Seaborn
- OpenCV

## 📦 Features
- CNN-based deep learning model with 94% accuracy
- Enhanced retinal image clarity by 30% using preprocessing
- Data augmentation increasing dataset size by 60%
- InceptionV3 architecture reducing training time by 25%
- Evaluated using precision, recall, AUC-ROC with 92% F1-score

## 🧠 Model Architecture
- Base Model: InceptionV3 (pretrained on ImageNet)
- Custom classification layers on top
- Binary classification: Diabetic Retinopathy vs Normal
- Optimized with Adam optimizer

| Metric | Score |
|--------|-------|
| Accuracy | 94% |
| F1-Score | 92% |
| AUC-ROC | 0.96 |
